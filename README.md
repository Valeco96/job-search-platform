# Job search platform - with role and location filter

## Description
Simple one page layout presenting a filter for job hunting. The filter has two categories, one for the role name and another one for the location of the job. It is possible to see the results according to what is included in the APIs, and to clean the search once it is over. Built during the Epicode Web Developer course. 

## Used technologies:
- HTML5
- CSS3
- JavaScript
- API Integration
